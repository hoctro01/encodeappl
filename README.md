# 🛡️ Tool Mã Hoá / Giải Mã Văn Bản (`encodeappl`)

![PyInstaller](https://img.shields.io/badge/PyInstaller-✔️-green)
![Platform](https://img.shields.io/badge/MacOS-arm64%20%7C%20x86--64-blue)
